# Notice
This is a mobile app. Please enter the developer option to select the mobile app for use

# Dev
```
$ git clone git@github.com:thosewhoyoulove/music.git
$ cd music
$ pnpm i # If you don't have pnpm installed, run: npm install -g pnpm
$ pnpm dev
```

# Build
```
$ pnpm build
```
